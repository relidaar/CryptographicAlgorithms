This project contains implementations of several symmetric/asymmetric cryptographic algorithms: from the ancient Caesar's cypher to the AES and RSA. 

### Currently supported algorithms

#### Substitution ciphers

- Caesar Cipher
- Polybius Cipher
- Vigenere Cipher
- Playfair Cipher
- Homophonic Cipher
